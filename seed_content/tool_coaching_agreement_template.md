# 教练协议模板

> ICF 第 3 项核心能力「建立与维护契约」的实操延伸。每个新客户必签的契约模板，可直接复制改动后使用。

## 为什么需要协议

不签协议直接做教练 = 自己埋雷：
- 没说清频次/时长 → 客户后期挑剔
- 没说清费用 → 收款扯皮
- 没说清边界 → 客户把教练当心理咨询
- 没说清取消政策 → 时间被反复浪费

签协议本身就是 **CC3 建立契约**的核心动作。

## 协议模板（可改）

```
教练协议

通过签署本协议，教练方、客户方共同认可以下条款。

【1. 收费】
本次教练旅程总费用为 ¥_______。

【2. 付费方式】
客户在签署协议后、下次教练会谈开始之前一次性支付。
(可改：按月预付 / 按次结算)

【3. 会谈安排】
- 总会谈次数：_____ 次
- 频次：_____ 次 / 月
- 时长：60 分钟 / 次（推荐 ICF 标准）
- 具体时间：双方事先约定

【4. 变更政策】
任何一方需调整预约，至少提前 24 小时通知。
推迟的会谈应在原约定时间后两周内完成。
(超过 48 小时通知 → 教练有权按 50% 收费 / 24 小时内取消 → 全额收费)

【5. 间期支持（教练增值）】
会谈间客户可临时联系教练（5-15 分钟内），不另收费。
此条款防止客户"囤问题等会谈" + 增强客户体验。

【6. 反馈机制】
客户对教练任何感受请及时提出（教练 CC1 伦理 + CC4 信任的明文承诺）。

【7. 保密】
教练承诺：
- 不向任何第三方泄露教练对话内容
- 不直接或间接利用客户信息为自己谋利
例外：法律强制披露（如自杀风险/重大刑事）—— 此情况必须明示。

【8. 关系边界（防混淆）】
本教练关系 **不是** 心理咨询 / 心理治疗 / 督导 / 管理建议。
客户对自己的决定和结果负责。
如客户过去被诊断有心理疾病，需在首次会谈告知。

【9. 终止】
首期教练至少 3 个月。3 个月后客户可决定继续或终止，
终止必须**书面**通知（邮件即可）。

【10. 录音】（CC5 / CC6 用于 ICF 申请）
本教练对话 (将 / 不将) 进行录音。
录音仅用途：（）教练自我反思（）ICF 申请提交（）双方共享。
录音 (将 / 不将) 在 90 天后删除。

客户签名：________________________ 日期：__________
教练签名：________________________ 日期：__________
```

## 改协议时注意

- ✏️ **收费政策要具体**：含税不含税、退款条件、改约扣费政策
- ✏️ **第 8 条边界声明很重要**：很多教练吃过亏的地方——客户把教练当心理咨询，教练越权处理
- ✏️ **录音条款分清楚**：ICF 申请、自我反思、和客户分享 → 三种用途三种授权方式
- ✏️ **GDPR / 数据合规**：欧盟客户加数据处理条款

## 签完之后的事

1. PDF 双方签字版存到加密云盘（不是普通微信）
2. 把协议**关键条款**口头复述一遍给客户确认（"我们刚签的协议，最重要的三件事是 A/B/C，你 OK 吗？"）—— 这是 CC3 的 "calibrate" 动作
3. 每次会谈开头确认当次目标（"今天 60 分钟你想聊什么？想达到什么？"）

## 与其他能力的关系
- **CC1 伦理**：第 7、8 条直接对应
- **CC3 契约**：本模板的核心
- **CC4 信任与安全**：边界清晰 = 安全感来源
