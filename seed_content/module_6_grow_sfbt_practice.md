## 💡 核心认知

### GROW vs SFBT 一表对比

| | GROW | SFBT |
|---|---|---|
| 时长 | 60 min | 30 min |
| 起点 | Goal 目标 | Solution 解决方案画面 |
| 焦点 | 行动计划 | 已有的资源 / 例外 |
| 经典提问 | "你想去哪？/你在哪？/有哪些路？/什么时候做？" | "奇迹问题 / 量尺 / 例外" |
| 适合 | 目标明确的高管对话 | 短期紧急议题 + 客户"卡住"感 |
| 节奏 | 结构化 | 流动 |
| 风险 | 太结构化压抑深度 | 太快错过深议题 |

### 怎么选？

```
客户来 → 问"你今天想聊什么"
   ↓
听 30 秒
   ↓
   ├─ "我想达成 X 目标" → GROW
   ├─ "我卡在 X" → SFBT
   ├─ "我不知道想要什么" → Co-Active / Erickson
   ├─ "我每次都重复同样的错误" → Gestalt
   └─ "我想理解自己" → Ontological / Co-Active
```

第一次见客户：**GROW 是最安全的选择**（结构最清晰，客户最有获得感）。
第 3-5 次会谈后熟悉客户：根据他的"卡点类型"选流派。

## 💪 实战练习

### 练习 1（自我）· 重读 + 提问清单

把 GROW 4 阶段的标志提问 + SFBT 3 大提问，**手写**在一张 A4 纸上（不是抄屏幕）。打印或贴在桌前。

这是你下次教练对话时的**外置脑**。

### 练习 2（伙伴）· 用 GROW 跑 60 分钟

找朋友，**议题约定为：他真实的某个职业/生活目标卡点**（不演戏）。

强制：
- 严格按 G(15min) → R(20min) → O(15min) → W(10min) 节奏
- W 阶段必须问"**从 1-10 你执行意愿是多少**"
- 完整录音

### 练习 3（同伴）· 同议题用 SFBT 跑 30 分钟

**几天后**（让议题"凉一下"），跟同一朋友，**同一议题**，用 SFBT 框架：

- 5 min 简短了解现状（不深挖）
- 10 min 奇迹问题 + 细节追问
- 10 min 量尺问题 + +1 分愿景
- 5 min 例外问题 + 下周行动

完整录音。

### 练习 4 · 用 /eval 对比两次评分 🎯

把练习 2 + 练习 3 的对话**分别**贴到 [/eval](/eval) 跑评估。

**鳌虾视角的对比分析**（写 300 字）：
- 两次 overall_level 一样吗？谁更高？
- 客户在两次会谈结束时**承诺的行动**有什么不同？
- 你作为教练，哪次"更累"？为什么？
- 哪次对话客户**自己发现了之前没意识到的东西**？

这就是 PCC 工头的核心能力——**知道用哪个流派+为什么**。
