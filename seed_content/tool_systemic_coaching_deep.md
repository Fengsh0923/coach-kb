# 系统教练 Systemic Coaching 深度指南

> "你不能在个体层面解决系统问题。" —— 这是系统教练的灵魂。家庭、团队、组织、社会都是**有机系统**，问题不是个人毛病，是系统模式的**涌现属性**。系统教练把客户**放进系统**理解，不孤立看。

## 起源

### 理论根（一般系统论）

**Ludwig von Bertalanffy**（1901-1972，奥地利-加拿大生物学家）
- **一般系统论**（General System Theory）创立者
- 把"系统作为整体"的视角应用到人类行为
- 1968 年：《一般系统论》出版

**Gregory Bateson**（1904-1980，英国人类学家）
- 把系统思维应用到家庭、文化、生态
- 控制论、双重束缚理论（double bind）的奠基者

**Roger Barker**（1903-1990，美国心理学家）
- 生态心理学（ecological psychology）先驱

### 家族治疗源流

**Murray Bowen**（1913-1990，美国精神病学家）
- **Bowen 家族系统理论**：核心概念**自我分化**（differentiation of self）
- 家庭被视为**情绪单元**，个人病症是家庭模式的表达

**Salvador Minuchin**（1921-2017，阿根廷-美国精神病学家）
- **结构家族治疗**（Structural Family Therapy）
- 关注家庭**结构 / 边界 / 联盟**

**Virginia Satir**（1916-1988，美国家族治疗师）
- 体验式家族治疗，**沟通模式**研究
- 后被 NLP（Bandler & Grinder）建模 — 见 [`tool_nlp_coaching_deep`](/wiki/tool_nlp_coaching_deep)

### 组织教练源流

**Peter Hawkins**（英国，组织咨询师）
- 现代**系统教练**（Systemic Coaching）领军人
- 著作 *Systemic Coaching: Delivering Value Beyond the Individual* (2019)

## 核心理念

### 1. 整体大于部分之和
个人行为**不能脱离系统理解**。同一个人在不同系统里（家庭 / 公司 / 朋友圈）行为模式不同——这不是"人格分裂"，是**系统作用**。

### 2. 问题是涌现属性
> "家庭问题被视为**系统互动的涌现属性**，而非可被归咎到个别成员的事。" — Wikipedia 系统心理学条目

**关键转变**：
- 传统："这个老板太奇葩"
- 系统视角："这个老板 + 这个团队 + 这个组织文化 → 共同产生当下的张力"

教练应用：客户抱怨"我老板是问题" → **把客户重新放回系统**问"你和老板**互相**怎么塑造了当下？"

### 3. 反馈循环 Feedback Loops
系统通过**正反馈**（放大）和**负反馈**（平衡）维持自身。理解循环 = 找到改变的杠杆。

### 4. 恒定性 Homeostasis
**系统倾向于维持现状**——即使现状不健康。这就是为什么"一个人努力改变家庭"通常失败——其他成员会**无意识地把他拉回来**。

教练应用：客户做出改变后家人朋友"突然不支持"——不是他们坏，是**系统在恢复恒定**。

## 系统教练 5 大工作动作

### 1. 把"问题"放回系统
- 客户："我跟下属沟通不畅。"
- 教练（个体派）："你试试 X 沟通技巧。"
- 教练（**系统派**）："这个'沟通不畅'**在你和下属之间，由你们共同维持**——你的什么部分**贡献**了这个模式？"

### 2. 寻找反馈循环
- 「你越 ___，他就越 ___，于是你又 ___」—— 这是一个正反馈循环
- 找到循环 = 找到打破点（在循环中任一节点改变就够了）

### 3. 边界 Boundaries 工作（Minuchin 派）
组织 / 家庭中的健康边界：
- **僵硬边界**（rigid）：信息/感情不能流动 → 疏离
- **过度渗透边界**（diffuse / enmeshed）：每个人都在他人事中 → 缠绕
- **健康边界**：清晰但渗透——既独立又联结

教练应用：让客户**画出**他在工作 / 家庭中的边界图（谁离他多近、信息怎么流）。

### 4. 系统排列（Hellinger 学派，争议技术）
**Bert Hellinger** 创立的"家庭系统排列"（Family Constellations）——让客户在房间里**用代表者站位**代表家庭成员，观察系统动力。

⚠️ **争议**：学术圈对 Hellinger 方法学有大量批评，不被主流系统教练界接受为**科学**方法。教练用时要明确标注**这是体验式工具，不是科学**。

### 5. 多层次工作（Hawkins 派）
任何客户都同时在 **5 个嵌套系统**中：
- 个人（自己）
- 团队（直接同事）
- 组织（公司）
- 行业（生态）
- 社会（更大文化）

教练应用：客户来谈"职业焦虑"——同时探索这 5 层在他身上的作用。

## 系统教练 vs 其他流派

| | Systemic | Ontological | Adlerian | CBT |
|---|---|---|---|---|
| 焦点 | 系统模式 | Being | 生活方式 | 想法 + 行为 |
| 单位 | 系统（关系网）| 个体 Being | 个体 + 社会兴趣 | 个体认知 |
| 见效速度 | 慢（理解系统需要时间）| 慢 | 中 | 快 |
| 适合 | 团队 / 组织 / 家庭 | 个人转型 | 反复模式 | 焦虑 / 行为 |

## 适合谁

✅ **管理者/领导力发展**（必然涉及团队系统）
✅ **企业内部教练**（个人 + 团队 + 组织三层）
✅ **创始人/合伙人冲突**（合伙关系是 2 人系统）
✅ **家族企业接班**（家庭 + 企业双系统耦合）
✅ **跨文化转型**（涉及社会文化层）

❌ 客户的议题完全个人化（如"我想戒烟"）— 用 CBT/SFBT
❌ 急需结果（系统视角需要时间消化）

## 5 个常见误用

### ❌ 把"系统视角"当借口推卸个人责任
"这都是系统的错" — 这是逃避。
**正解**：系统视角 ≠ 个人无责——是**在系统中找到我的杠杆点**。

### ❌ 教练自己想"解开整个系统"
不可能。系统教练**只在客户的视角**工作——别人怎么样，客户不知道，**只能改变自己在系统中的位置**。

### ❌ Hellinger 系统排列当成"科学"
不科学。用时明确"这是体验式探索"。

### ❌ 跳过个人层直接做系统
"你妈妈的童年伤了你" — 教练不该挖客户家庭历史。系统视角是工具，不是治疗。

### ❌ 系统大到让人无力
"问题在中国教育系统" — 客户能改变中国教育吗？聚焦在**他能影响的范围**。
