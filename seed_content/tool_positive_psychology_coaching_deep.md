# 积极心理学教练 Positive Psychology Coaching 深度

> 积极心理学不是"鸡汤要乐观"——它是 **Martin Seligman 1998 年发起的科学心理学子学科**，研究"什么让人**蓬勃发展**"。教练界用它的模型（PERMA / VIA 优势 / 心流）做**优势导向**而不是问题导向的教练。

## 起源

**Martin Seligman**（宾大教授）在 **1998 年**担任美国心理学会（APA）主席期间，选择"积极心理学"作为任期主题。

**Seligman 的诊断**：
> "主流心理学过度关注疾病与障碍，忽视了幸福、韧性与复原力。我的目标是**应用主流心理学的方法论和科学优势来促进幸福**，而非聚焦疾病。"

**为什么这重要**：传统心理学问"你有什么病"——积极心理学问"你有什么优势可以放大"。教练用这个角度问客户：「**你的优势是什么**？怎么让它在工作里更多发挥？」

## 3 大核心模型

### 1. PERMA（幸福 5 要素）

Seligman 在 *Flourish*（2011）提出：

| 要素 | 含义 | 教练用法 |
|---|---|---|
| **P**ositive Emotions | 积极情绪 | "本周哪些时刻你感到喜悦/感激/平静？" |
| **E**ngagement | 全神贯注/心流 | "什么活动让你忘记时间？" |
| **R**elationships | 关系 | "谁让你感到被看见？" |
| **M**eaning | 意义/目的 | "你做的事如何超越自己？" |
| **A**ccomplishments | 成就 | "本周你为什么自豪？" |

**教练应用**：客户陷入"我很丧" → 教练问 PERMA 5 个维度的当前分数 → 找出**最低**的去工作，或找出**最高**的去放大。

### 2. VIA 性格优势（6 美德 + 24 优势）

Peterson & Seligman *Character Strengths and Virtues*（2004）识别了**跨文化通用**的 24 个性格优势，分 6 大美德：

| 美德 | 包含的优势示例 |
|---|---|
| **智慧 Wisdom** | 创意 / 好奇心 / 判断力 / 求知欲 / 洞察 |
| **勇气 Courage** | 勇敢 / 毅力 / 真诚 / 热情 |
| **人性 Humanity** | 爱 / 善良 / 社交智慧 |
| **正义 Justice** | 团队合作 / 公平 / 领导力 |
| **节制 Temperance** | 宽恕 / 谦虚 / 谨慎 / 自我调节 |
| **超越 Transcendence** | 审美 / 感恩 / 希望 / 幽默 / 灵性 |

**教练应用**：
- 让客户上 [viacharacter.org](https://www.viacharacter.org) **免费**测自己的 VIA 优势排序
- 引导客户**用自己的优势**解决问题，而不是补短板
- 「你的 Top 5 优势里，**勇气**怎么帮你应对当下这个冲突？」

⚠️ **关键转变**：积极心理学**反对"修补短板"**模式，主张"放大优势"。这与中国教育系统从小灌输的"哪里弱补哪里"完全相反——这正是国内高管最缺的视角。

### 3. 心流 Flow（Csikszentmihalyi）

**Mihaly Csikszentmihalyi**（米哈伊·契克森米哈伊）提出**心流**：

**心流是"最优体验"，4 大特征**：
- 高度**专注**与沉浸
- **挑战与技能平衡**（不能太容易也不能太难）
- 失去**自我意识感**（"忘了自己"）
- **时间飞逝感**

**心流产生条件**：挑战水平**正好匹配**个人能力——略高一点（让人需要伸展），但不会高到放弃。

**教练应用**：
- 「上次你**忘记时间**的工作是什么？」（找心流活动）
- 「你目前任务的挑战 vs 技能匹配怎么样？」（诊断焦虑/无聊）
- 设计每周**至少 1 次心流体验**

## 积极心理学教练的 60 分钟节奏

```
0-5    契约 + 这周积极时刻盘点（不直接进问题）
5-15   PERMA 5 维度速评（哪几个低？哪个高？）
15-30  深入低分维度 + 探索"低的时候你的什么优势可以用上"
30-45  用 VIA Top 5 优势设计行动
45-55  心流：本周设计 1 次 90 分钟心流体验
55-60  庆祝 + 收尾
```

⚠️ 注意：积极心理学**不是无视问题**——它在承认问题的前提下问"什么力量可以应对"。如果客户陷入危机/抑郁，**应转介专业心理治疗**，不能强推积极。

## 6 个常见误用

### ❌ "假装积极"
让客户硬挤"感激清单"——这是表演不是真心。
对：从客户**真实**经历里找他**真心**感激的小事。

### ❌ 忽视真实痛苦
客户痛苦 → 教练立刻问"积极的方面是什么"——这是**回避**。
对：先承认痛苦的真实，再问"在痛苦里你的什么部分还活着？"

### ❌ "强加 VIA 测试"
客户没问就让他测——侵入。
对：在客户卡住时邀请测试作为"重新看自己"的工具。

### ❌ 心流变工具崇拜
"你必须每周心流 5 次"——这是绩效压力。
心流是**自然涌现**的副产品，不是 KPI。

### ❌ PERMA 当问卷打分
形式化打分跟讨论实际困境无关。
PERMA 是**视角**不是测试。

### ❌ 与抑郁症客户硬上
临床抑郁需要专业治疗 + 可能药物。教练用积极心理学 ≠ 治疗。**红线**：识别危机信号转介。

## 适合谁

✅ **职业转型者**：找自己的优势重新定位
✅ **倦怠的中层**：用心流和 PERMA 重新连接工作意义
✅ **新晋管理者**：用 VIA 优势认识团队
✅ **想升级生命质量**的人（不是缓解病情）

❌ 处于**严重抑郁/焦虑**：转介心理治疗
❌ **危机情境**（自杀/暴力风险）：报警/转介危机服务
❌ **创伤未处理**：用 EMDR/Somatic Experiencing 等专业方法
